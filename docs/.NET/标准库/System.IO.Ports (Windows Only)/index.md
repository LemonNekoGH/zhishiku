# `SerialPort`类

</p>

<p id="wppGj3kTs4gECkVjxsa5PR">

### 事件

</p>

- `DataReceived`指示已通过由`SerialPort`对象表示的端口接收了数据。

<p id="kEtJdPhZ5os9XFnAwrVBsT">

### 属性

</p>

- `ReceivedBytesThreshold`事件触发前内部输入缓冲区中的字节数。 默认值为 1。<span style="color:#E91E2C;">**一定要改。**</span>
